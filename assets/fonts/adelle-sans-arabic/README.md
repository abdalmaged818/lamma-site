# Adelle Sans Arabic — Font Files

This folder is the upload location for the official "Adelle Sans Arabic" brand font files.

The CSS at `assets/css/fonts.css` already contains @font-face rules that point to the file names listed below. Upload each weight using these exact file names (both .woff2 and .woff formats if available) so the fonts work without any further code changes.

| Weight | Style Name | Expected file name (base) |
|--------|------------|----------------------------|
| 100 | UltraThin | AdelleSansArabic-UltraThin |
| 200 | Thin | AdelleSansArabic-Thin |
| 300 | Light | AdelleSansArabic-Light |
| 400 | Regular | AdelleSansArabic-Regular |
| 600 | SemiBold | AdelleSansArabic-SemiBold |
| 700 | Bold | AdelleSansArabic-Bold |
| 800 | ExtraBold | AdelleSansArabic-ExtraBold |
| 900 | Heavy | AdelleSansArabic-Heavy |

For each row, upload files named `<base>.woff2` and `<base>.woff` (e.g. `AdelleSansArabic-Regular.woff2`, `AdelleSansArabic-Regular.woff`) directly into this folder.

Once the real font files are uploaded here, the site can start using `font-family: "Adelle Sans Arabic";` anywhere in the CSS — no folder or file name changes will be required.
